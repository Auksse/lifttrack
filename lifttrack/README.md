# LiftTrack

Personal workout tracking PWA — built with vanilla JS, Supabase, hosted on GitHub Pages.

**Live:** https://auksse.github.io/lifttrack/

## Stack
- Frontend: Vanilla JS, single HTML file
- Database: Supabase (Postgres + REST API)
- Hosting: GitHub Pages

## Setup
1. Clone the repo
2. Open the app, enter your Supabase URL + anon key
3. Done — data syncs automatically

## Development
Edit `index.html` directly. Push to `main` and GitHub Pages auto-deploys in ~30s.
